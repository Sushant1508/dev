1. Update your npm using cmd/terminal 
	npm install npm@latest -g

2. Run the below command in your root of project.

npm i  -D babel-core babel-loader babel-preset-es2015 babel-preset-react react-hot-loader webpack webpack-dev-server
npm i webpack webpack-dev-server -g

3. When all packages are downloaded, run "webpack-dev-server" in your project root. It should start serving your webpage on http://localhost:8080
